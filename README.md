# User Registration and Authentication System
**IT342 - Laboratory 2**

## Project Overview
This project implements a secure User Registration and Authentication system using a Spring Boot backend and a ReactJS web application. It features password encryption via BCrypt and session management using JWT.

## Technologies Used
* **Backend:** Java 17, Spring Boot, Spring Security, JPA, MySQL
* **Frontend:** ReactJS, Axios, React Router
* **Documentation:** FRS, UML (Class, Sequence, Activity Diagrams), ERD

## Folder Structure
* `/backend`: Spring Boot API
* `/web`: ReactJS Web Application
* `/docs`: Project documentation and FRS PDF
* `/mobile`: Mobile application (Planned for Session 2)

## Setup Instructions
1. Configure MySQL database in `backend/src/main/resources/application.properties`.
2. Run the Spring Boot application.
3. Run `npm install` and `npm start` in the `/web` directory.
